
## OverView

The User Directory Application is a frontend-only React application that fetches user data from a public API and displays it in a clean, responsive interface. Users can search for users, view detailed information, and add new users using a validated form.

This project demonstrates modern frontend practices such as component-based architecture, state management using React Hooks, form validation, and responsive UI design with Tailwind CSS.

## Tech Stack
.React
.tailwind css
.javascript
